Fake Discount and Price History Verification System

# Overview
The Fake Discount and Price History Verification System is a web-based application developed to help online shoppers identify genuine and misleading discounts on Amazon products. The system extracts product information such as price, MRP, rating, and review count using web scraping techniques and analyzes the data using a Random Forest Machine Learning model.
Based on the analysis, the system classifies products as Safe, Moderate, or Risky, enabling users to make informed purchasing decisions and avoid deceptive pricing practices.
